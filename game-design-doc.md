## 一、核心差异化定位

### 1.1 东方末世题材

**为什么选择东方题材？**

| 对比维度 | 西方魔幻（躺平发育） | 东方玄幻（灵木求生） | 优势 |
|----------|---------------------|---------------------|------|
| **题材独特性** | 吸血鬼/哥特 | 仙侠/末法时代 | ✅ 避免同质化 |
| **目标用户** | 全球市场 | 东亚+东南亚 | ✅ 文化认同 |
| **美术风格** | 暗黑哥特 | 水墨像素风 | ✅ 差异化美感 |
| **付费意愿** | 中等 | 高（修仙文化） | ✅ 付费潜力 |
| **社交传播** | 中等 | 高（仙盟/门派） | ✅ 话题性 |

### 1.2 末法时代设定

```
【世界观背景】

天地巨变，末法降临
├── 灵气枯竭，仙道断绝
├── 妖兽横行，人族式微
├── 灵木变异，成为最后希望
└── 修仙者转修灵木之法

玩家身份：灵木师（修仙末代传承者）
核心目标：培育灵木，重建修仙文明
```

### 1.3 三大玩法融合

```
【玩法融合矩阵】

                    灵木求生
                       │
       ┌───────────────┼───────────────┐
       │               │               │
   ┌───┴───┐      ┌────┴────┐      ┌───┴───┐
   │ 仙府  │      │  妖兽  │      │ 灵兽  │
   │ 建造  │  +   │  塔防  │  +   │ 契约  │
   │       │      │       │      │       │
   │ 仙木  │      │ 妖潮  │      │ 灵宠  │
   │ 升级  │      │ 抵御  │      │ 养成  │
   └───────┘      └───────┘      └───────┘

融合亮点：
├── 仙木建造 → 仙府布置（建造）
├── 阵法防御 → 妖兽抵御（塔防）
├── 灵兽助战 → 仙宠协同（宠物）
└── 末法修仙 → Roguelike（随机）
```

## 二、东方元素体系

### 2.1 核心概念对照

| 原概念（西式） | 东方概念 | 说明 |
|---------------|----------|------|
| 树屋 | **灵木/仙木** | 变异灵树，修炼根基 |
| 采集 | **采灵/炼丹** | 采集灵气，炼制丹药 |
| 建造 | **仙府/洞府** | 修仙者的居所 |
| 防御 | **阵法/符箓** | 东方特有的防御手段 |
| 怪物 | **妖兽/魔物** | 末法时代的妖邪 |
| 宠物 | **灵兽/仙宠** | 可契约的灵性生物 |
| 装备 | **法器/灵器** | 修仙者的武器 |
| 货币 | **灵石/灵晶** | 灵气凝结的货币 |
| 科技 | **功法/秘术** | 修仙功法 |
| 技能 | **法术/神通** | 修仙者的能力 |

### 2.2 美术风格定位

```
【视觉风格：水墨像素风】

风格关键词：
├── 水墨渲染 → 飘逸灵动的视觉效果
├── 像素精绘 → 精致的东方角色像素
├── 朱砂点缀 → 中国红作为强调色
├── 仙气缭绕 → 灵气粒子特效
└── 古典建筑 → 东方仙侠元素

配色方案：
├── 主色：墨色（#1a1a2e）→ 玄黑
├── 辅色：青白（#e8f4f8）→ 仙气
├── 强调：朱砂（#d63031）→ 战斗
├── 灵气：青绿（#00b894）→ 治愈
└── 灵木：金色（#fdcb6e）→ 升级
```

### 2. 3 UI设计语言

```
【UI风格：古典仙侠风】

界面框架：
├── 边框：水墨卷轴风格
├── 图标：玉佩/灵石/灵草造型

├── 背景：山水云雾纹理
```

## 三、核心系统强化

### 3.1 灵木（树屋）升级体系

```
【灵木等级设计】

Lv.1 灵种初萌
├── 形态：刚发芽的灵种
├── 解锁：基础储物
└── 特效：微光闪烁

Lv.2 幼苗初成
├── 形态：筷子高的幼苗
├── 解锁：炼丹炉
└── 特效：嫩叶摇曳

Lv.3 小灵木
├── 形态：一人高的小树
├── 解锁：防御阵眼×2
└── 特效：灵气缭绕

Lv.4 中灵木
├── 形态：参天大树
├── 解锁：灵兽窝
└── 特效：枝叶繁茂

Lv.5 仙木
├── 形态：灵光万丈
├── 解锁：仙府核心
└── 特效：瑞兽来朝

Lv.6 灵根（终级）
├── 形态：世界树级别
├── 解锁：终极技能
└── 特效：改天换地
```

```

├── 九宫阵：攻防兼备
└── 天罗阵：范围禁锢

高级防御（灵器）：
├── 昊天镜：反伤反射
├── 万剑诀：万剑齐发
├── 玲珑塔：绝对防御
└── 翻天印：全屏AOE

终极防御（神通）：
├── 天雷劫：连续雷击
├── 三昧真火：持续燃烧
└── 移山填海：改变地形
```
