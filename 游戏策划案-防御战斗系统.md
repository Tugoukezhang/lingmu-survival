## 一、系统概述

### 1.1 系统定位

妖兽抵御系统是游戏的**核心玩法系统**，对应《猛鬼宿舍》的塔防机制，但以**东方阵法符箓**为核心，提供**紧张感**、**策略深度**和**东方战斗美感**。

### 1.2 设计目标

| 目标 | 描述 | 东方元素 |
|------|------|----------|
| 制造紧张感 | 妖潮进攻的压迫感 | 妖兽围城，天雷滚滚 |
| 策略深度 | 阵法搭配的策略性 | 五行相克，八卦阵法 |
| 反馈即时性 | 战斗结果的即时反馈 | 道纹发光，法术特效 |
| 数值成长 | 法器阵法的强化 | 渡劫提升，炼器加成 |

### 5.3 难度曲线（修仙难度）

```
【难度曲线设计 - 修仙劫难】

难度等级 = BaseDifficulty × WaveMultiplier × DemonMultiplier × TianDaoMultiplier

BaseDifficulty = 1.0
WaveMultiplier = 1 + (waveNumber - 1) × 0.25
DemonMultiplier = 1.0 + demonEliteRate
TianDaoMultiplier = 1.0 + tribulationBonus

境界难度调整：
├── 炼气期：难度×0.8（新手法师）
├── 筑基期：难度×1.0（正常难度）
├── 金丹期：难度×1.5（精英挑战）
└── 元婴期：难度×2.0（天道考验）

示例：
├── 第1波（炼气）：1.0 × 1.0 × 1.0 = 1.0
├── 第5波（筑基）：1.0 × 2.0 × 1.0 = 2.0
├── 第5波+妖王：1.0 × 2.0 × 1.5 = 3.0
└── 金丹妖王：1.0 × 2.5 × 2.0 = 5.0
```

### 9.1 阵法AI

```javascript
// 阵法攻击AI
class FormationAI {
  constructor(formation) {
    this.formation = formation;
    this.target = null;
    this.element = formation.element;
  }
  
  // 攻击逻辑
  update(enemies) {
    // 1. 找目标（优先被克属性）
    this.target = this.findTarget(enemies);
    
    // 2. 攻击
    if (this.isInRange(this.target)) {
      this.attack(this.target);
    }
    
    // 3. 触发特效
    this.triggerEffect();
  }
  
  // 寻找目标（五行相克优先）
  findTarget(enemies) {
    // 优先攻击被克制的妖兽
    const weakEnemies = enemies.filter(e => 
      this.isWeakAgainst(e.element)
    );
    
    if (weakEnemies.length > 0) {
      return this.findNearest(weakEnemies);
    }
    
    return this.findNearest(enemies);
  }
  
  // 五行克制判断
  isWeakAgainst(targetElement) {
    const weakness = {
      'Metal': 'Wood',
      'Wood': 'Earth', 
      'Earth': 'Water',
      'Water': 'Fire',
      'Fire': 'Metal'
    };
    return weakness[this.element] === targetElement;
  }
}
```
