## 一、系统概述

### 1.1 系统定位

灵木仙府系统是游戏的**核心成长系统**，融合东方修仙文化，通过培育灵木、建造仙府，为玩家提供**目标感**、**成就感**和**文化认同感**。

### 1.2 设计目标

| 目标 | 描述 | 东方元素 |
|------|------|----------|
| 提供成长目标 | 玩家有明确的渡劫方向 | 灵木渡劫，羽化成仙 |
| 增加沉没成本 | 让玩家对仙府产生情感 | 仙府养成，传承感 |
| 解锁新功能 | 高等级解锁新玩法 | 境界突破，法术解锁 |
| 视觉反馈 | 逐步壮大的成就感 | 灵光万丈，瑞兽来朝 |

### 8.2 渡劫系统

```javascript
// 渡劫流程
class TribulationSystem {
  constructor(player) {
    this.player = player;
    this.currentTrial = null;
    this.trialProgress = 0;
  }
  
  // 计算渡劫成功率
  calculateSuccessRate() {
    let rate = this.getBaseRate();
    rate += this.getBeastBonus();
    rate += this.getMeritBonus();
    rate += this.getArtifactBonus();
    rate += this.getFengShuiBonus();
    return Math.min(rate, 95); // 最高95%
  }
  
  // 开始渡劫
  async startTribulation() {
    const rate = this.calculateSuccessRate();
    const roll = Math.random() * 100;
    
    if (roll < rate) {
      await this.runTrials();
      return await this.success();
    } else {
      await this.runTrials();
      return await this.failure();
    }
  }
  
  // 试炼阶段
  async runTrials() {
    // 第一道：心魔试炼（心理博弈）
    await this.trialHeartDemon();
    
    // 第二道：五行雷劫（战力考验）
    await this.trialLightning();
    
    // 第三道：天人交战（极限挑战）
    await this.trialHeavenMan();
  }
}
```
