## 一、项目概述

### 1.1 项目定位

《灵木求生》是一款**东方玄幻末世求生游戏**，改编自起点小说《求生之开局一个小树屋》，融合**塔防建造**+**Roguelike**+**放置养成**三种玩法，主打**东方修仙文化**差异化定位。

### 1.2 核心玩法

```
【三合一融合玩法】

                    灵木求生
                       │
       ┌───────────────┼───────────────┐
       │               │               │
   ┌───┴───┐      ┌────┴────┐      ┌───┴───┐
   │ 灵木  │      │  妖兽  │      │ 灵兽  │
   │ 建造  │  +   │  塔防  │  +   │ 契约  │
   │       │      │       │      │       │
   │ 渡劫  │      │ 阵法  │      │ 问道  │
   │ 升级  │      │ 抵御  │      │ 养成  │
   └───────┘      └───────┘      └───────┘
```

### 1.3 题材优势

| 维度 | 西方魔幻（竞品） | 东方玄幻（灵木求生） | 优势分析 |
|------|------------------|----------------------|----------|
| **题材独特性** | 吸血鬼/哥特 | 仙侠/末法时代 | ✅ 避免同质化 |
| **文化认同** | 西方 | 东亚+东南亚 | ✅ 认同感强 |
| **付费意愿** | 中等 | 高（修仙文化） | ✅ 付费潜力 |
| **社交传播** | 中等 | 高（仙盟/门派） | ✅ 话题性强 |
| **IP延伸** | 弱 | 强（仙侠文化） | ✅ 可持续发展 |

## 五、商业化设计

### 5.1 付费架构

```
【付费设计 - 健康修仙生态】

核心原则：
├── 不逼氪：核心内容免费体验
├── 不逼肝：离线收益保障
└── 快乐修仙：外观+收集为主

付费结构：
├── 广告收入：55%
│   ├── 激励视频：看广告加速
│   ├── 插屏广告：日常展示
│   └── 开屏广告：启动展示
│
└── 内购收入：45%
    ├── 月卡（仙缘卡）：35%
    ├── 战令（渡劫战令）：30%
    ├── 灵兽契约：25%
    └── 外观/装饰：10%

付费点设计：
├── 【月卡】仙缘卡（30元/月）
│   ├── 每日100灵晶
│   ├── 经验+20%
│   └── 渡劫保护×1/日
│   └── 价值：性价比极高
│
├── 【战令】渡劫战令（68元/季）
│   ├── 限定灵兽×1
│   ├── 限定法器套装
│   ├── 专属特效
│   └── 仙缘×500
│   └── 价值：内容丰富
│
├── 【直购】灵兽契约（10-328元）
│   ├── 凡品契约：10灵晶
│   ├── 上品契约：68灵晶
│   ├── 天命契约：328灵晶
│   └── 价值：收集向
│
└── 【外观】仙府皮肤（30-128元）
    ├── 灵木外观皮肤
    ├── 法器特效皮肤
    └── 阵法光环皮肤
    └── 价值：个性化
```

### 5.2 禁止付费内容

```

├── 外观装饰（个性化）

✅ 免费玩家权益：

├── 全部天道事件

```
