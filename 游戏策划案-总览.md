| 灵木视觉 | 6级描述 | **7级视觉进化** | 大小/光效/粒子/符文圈/光环递增 |

## 三、系统关联（修仙体系）

### 3.1 系统依赖关系

```
【灵木仙府 → 妖兽抵御】
├── 灵木境界 → 解锁阵法槽位（Lv1=2/Lv3=3/Lv5=4/Lv7=5/Lv9=6）
├── 灵木等级 → 阵法伤害加成（每级+8%）
├── 灵木渡劫 → 阵法解锁更高阶形态
└── 灵木视觉进化 → 7级形态递增反馈

【妖兽抵御 → 灵木仙府】
├── 战斗奖励 → 灵木升级材料
├── 妖王掉落 → 稀有渡劫材料
├── 波次奖励 → 灵石/经验
├── 完美防守 → 天道眷顾（额外奖励）
└── 战绩评级 → D-SSS结算奖励

【天命劫难 → 灵木仙府】
├── 天命事件 → 材料加成/减益
├── 问道奖励 → 灵根材料/灵石
├── 功法效果 → 永久属性加成
├── 渡劫系统 → 境界突破（Lv3风劫/Lv6火劫/Lv9雷劫）
└── 功德积累 → 渡劫成功率

【天命劫难 → 妖兽抵御】
├── 天命事件 → 战斗增益/减益
├── 昼夜时段 → 影响阵法/妖兽数值
├── 功法加成 → 阵法/灵兽强化
└── 符箓使用 → 战斗即时技能
```

### 3.2 资源共享（修仙资源）

```
【修仙资源】

境界资源：
├── 灵气：基础成长资源（破晓×1.5回复）
├── 灵石：通用战斗货币
├── 灵矿：炼器次要材料
├── 灵晶：稀有付费货币

└── 仙缘：功法永久货币

共享存储：
├── 储物袋：所有物品
├── 仙府仓库：资源存储
├── 灵兽栏位：灵兽存放
└── 功法阁：功法技能

东方特色：
├── 道心：心理考验指标
├── 命格：命运随机属性
├── 天道：全局运气系统
└── 轮回：失败重修机制
```

## 六、付费设计（修仙生态）

### 6.1 付费点设计

```
【付费结构 - 健康修仙生态】

付费理念：
├── 不逼氪：核心内容免费
├── 不逼肝：离线收益保障
└── 快乐修仙：外观+收集为主

付费点类型：
├── 【月卡】仙缘卡（30元/月）
│   ├── 每日100灵晶
│   ├── 经验+20%
│   └── 渡劫保护×1/日
│
├── 【战令】渡劫战令（68元/季）
│   ├── 限定灵兽
│   ├── 限定法器
│   └── 专属特效
│
├── 【直购】灵兽契约（10-328元）
│   ├── 凡品契约：10灵晶
│   ├── 上品契约：68灵晶
│   └── 天命契约：328灵晶
│
├── 【外观】仙府皮肤（30-128元）
│   ├── 灵木外观
│   ├── 法器皮肤
│   └── 特效光环
│
├── 【加速】修炼加速（6-68元）
│   ├── 灵木升级加速
│   ├── 阵法升级加速
│   └── 功法研习加速
│
└── 【复活】渡劫保护（6元/次）
    ├── 渡劫失败不降级
    └── 保留所有进度

收入结构目标：
├── 广告收入：55%
├── 内购收入：45%
│   ├── 月卡：35%
│   ├── 战令：30%
│   ├── 灵兽契约：20%
│   └── 外观+加速+保护：15%
```

```
【50波妖潮设计 - 5波一循环】

每5波为一循环：
├── 第1-4波：常规妖潮（递增难度）
├── 第5波：妖王降临（Boss波）
└── 完美防守（0漏怪）：额外灵石奖励

难度缩放：
├── 怪物HP：基础 × (1 + (wave-1) × 0.08)
├── 怪物数量：5 + wave × 2
├── 妖王血量：基础 × (1 + cycle × 0.5)
└── 夜间波次：妖兽攻击×1.5，阵法×0.7
```
