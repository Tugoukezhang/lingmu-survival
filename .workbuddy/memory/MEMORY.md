## 用户项目

### 游戏开发项目：《灵木求生》（原《求生之开局一个小树屋》）

**项目概述**：
- 基于起点小说《求生之开局一个小树屋》(作者:为何我是神)改编
- **v2.0版本：东方玄幻末世求生，强化差异化定位**
- 2D像素风+水墨风格融合类游戏
- 融合猛鬼宿舍(塔防) + 肉鸽(Roguelike) + 生存建造 + **东方修仙**

**核心玩法（v2.0强化）**：
- **灵木仙府系统**：树屋→灵木，融入渡劫/灵根/枝叶变异机制
- **妖兽抵御系统**：防御→阵法符箓，融入五行相克
- **天命劫系统**：随机事件→天道轮回，融入命格/道心/问道机制
- 昼夜修仙计时（子时妖潮/卯时天亮）
- 灵兽契约系统（仙宠养成）
- 仙盟社交系统（门派归属）

**差异化定位**：
- 东方仙侠题材（避免西方哥特同质化）
- 水墨像素美术风格
- 健康付费生态（不逼氪不逼肝）
- 修仙文化深度（渡劫/功德/道心）

**技术选型**：
- Phaser.js 3.x
- 微信小游戏 + Web双端
- AI辅助美术（Nano Banana Pro + 像素绘制）

**项目状态**：
- 可行性验证：✅ 通过
